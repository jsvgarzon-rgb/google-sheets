Caja de Herramientas

Domina estas cuatro áreas y tus hojas pasarán de ser tablas simples a poderosos analizadores de datos.

ƒ Fórmulas Esenciales

Las fórmulas son instrucciones que le das a Sheets para que calcule automáticamente. Siempre empiezan con = y trabajan con rangos de celdas.

  • =SUMA(A1:A10) — Suma todos los valores del rango
  • =PROMEDIO(A1:A10) — Calcula el promedio del rango
  • =SI(condición, "sí", "no") — Decisión automática
  • =CONTAR.SI(rango, criterio) — Cuenta con condición

Ejemplos en acción:

=SUMA(D2:D10)
→ Suma todas las notas del rango D2 a D10
=PROMEDIO(D2:D10)
→ Calcula el promedio de calificaciones
=SI(D2>=9, "Excelente", "Regular")
→ Categoriza automáticamente la nota
=CONTAR.SI(C2:C10, "Diseño")
→ Cuenta cuántas herramientas son de Diseño
📊 Crear Gráficos

Selecciona los datos, ve a Insertar → Gráfico y Sheets sugiere automáticamente el tipo más adecuado. Puedes personalizarlo desde el Editor de gráficos.

  • Selecciona el rango de datos primero (incluyendo encabezados)
  • Insertar → Gráfico para abrir el editor
  • Cambia colores y títulos en la pestaña "Personalizar"
  • El gráfico se actualiza automáticamente al cambiar los datos

🎨 Formato Condicional

El formato condicional colorea celdas automáticamente según reglas. Ve a Formato → Formato condicional y define cuándo una celda debe cambiar de color.

  • Selecciona el rango de celdas a formatear
  • Define la condición (mayor que, menor que, igual a...)
  • Elige el color de relleno o de texto
  • Agrega múltiples reglas para crear semáforos de datos
Verde → Nota ≥ 9.0 (Excelente)
Amarillo → Nota entre 7.0 y 8.9
Rojo → Nota < 7.0 (Revisar)

🗂️ Gestión de Datos

Sheets incluye herramientas potentes para organizar, filtrar y validar tus datos sin escribir una sola fórmula.

🔽
Ordenar
Datos → Ordenar hoja. Ordena por cualquier columna A→Z o Z→A
🔍
Filtros
Datos → Crear un filtro. Muestra solo las filas que necesitas
✅
Validación
Datos → Validación de datos. Evita errores de ingreso con listas desplegables
🔗
Tablas dinámicas
Insertar → Tabla dinámica. Resúmenes automáticos de grandes volúmenes de datos
⌨️ Atajos Esenciales

Trabaja 3 veces más rápido usando estos atajos de teclado en Sheets:

  • Ctrl+Shift+V — Pegar solo valores (sin fórmulas)
  • Ctrl+Shift+L — Activar/desactivar filtros
  • Ctrl+D — Copiar fórmula hacia abajo
  • Alt+Enter — Nueva línea dentro de una celda
  • Ctrl+Alt+M — Insertar comentario en celda
